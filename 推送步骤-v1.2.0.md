#### 方式B：使用命令行（如果Git已安装）

打开Git Bash或PowerShell，执行：

```bash
cd E:\media-renamer
git add .
git commit -m "feat: v1.2.0 - 实时日志推送和元数据查询优化"

```

### 步骤2：更新服务器

推送成功后，运行：

```powershell
.\远程推送更新.ps1
```

这个脚本会：
1. 检查服务器状态
2. 检查是否有更新
3. 自动执行更新
4. 重启服务
5. 验证服务状态
