# 基础配置

## 1. 用户安全
- **管理员账号**  
  默认: `admin`/`password`  
  ⚠️ 首次登录后必须修改密码！使用默认密码暴露公网有严重安全风险！

## 2. 网络代理
- **代理服务器**  
  格式示例:  
  - HTTP: `127.0.0.1:7890`  
  - SOCKS5: `socks5://127.0.0.1:8018`  
  - 认证代理: `socks5://user:pass@127.0.0.1:7891`  

  ℹ️ 默认仅用于TMDB/Telegram访问，站点需单独开启

## 3. 外网访问
- **访问地址**  
  格式: `http(s)://IP:端口`  
  ⚠️ Telegram Webhook需使用特定端口: 443/80/88/8443

# 媒体库配置

## 1. TMDB设置
- **API Key**  
  ℹ️ 必须配置，否则核心功能无法使用
- **匹配模式**  
  - 正常模式: 识别率高但可能误识别  
  - 严格模式: 准确率高但要求规范命名

## 2. 分类策略
```yaml
# 示例配置
movie:
  华语电影:
    original_language: 'zh,cn'
  外语电影:

tv:
  国产剧:
    origin_country: 'CN,TW,HK'
  欧美剧: 
    origin_country: 'US,UK'
```

ℹ️ 完整分类字典见[分类参考](#分类参考)

## 3. 文件处理
- **重命名格式**  
  支持占位符如`{title}`、`{year}`等  
  示例: `电影/{title} ({year})/{title}.{ext}`

- **刮削设置**  
  可生成nfo元数据和媒体图片  
  ⚠️ "演职人员中文"选项会频繁访问豆瓣

# 服务配置

## 1. 订阅设置
| 功能 | 建议值 | 说明 |
|------|--------|------|
| RSS周期 | ≥300秒 | 最小间隔300秒 |
| 搜索周期 | ≥6小时 | 对站点压力大 |

## 2. 同步设置
- **媒体库同步**  
  保持本地与媒体服务器状态一致  
- **站点刷新**  
  支持多种时间格式:  
  - 间隔: `12` (小时)  
  - 固定时间: `08:00`  
  - 时间范围: `08:00-09:00`  
  - Cron表达式: `0 */6 * * *`  

# 安全设置

## 1. 访问控制
- **源地址限制**  
  示例:  
  - Telegram IP: `149.154.160.0/20,91.108.4.0/22`  
  - 全部开放: `0.0.0.0/0,::/0`  

## 2. API安全
- **密钥验证**  
  启用后需在URL添加`?apikey=xxx`参数

# 高级功能

## 1. 识别增强
- **辅助识别**: 增加识别率但耗时  
- **ChatGPT识别**: 效果佳但需付费  

## 2. 搜索优化
- **豆瓣优先**: 中文支持好但可能不一致  
- **多语言搜索**: 提高资源匹配率

## 3. 辅助功能
- **验证码识别**: 添加验证码识别服务器，以支持站点签到
- **网页自动化服务器**: 部分站点需要配置仿真才能访问
