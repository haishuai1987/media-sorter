# 索引器配置

## 内建索引器
- 站点维护中添加站点后，支持的内建索引器会自动显示
- **只有选中的站点才会在搜索中使用**
- 新站点适配需求请在[nas-tools-sites项目](https://github.com/linyuan0213/nas-tools-sites)提issues

## 外置索引器
### Jackett
- **服务器地址**：Jackett服务IP和端口
- **API Key**：Jackett控制面板中获取
- **密码**：Jackett访问密码（如设置）

### Prowlarr
- **服务器地址**：Prowlarr服务IP和端口
- **API Key**：Prowlarr设置中获取
