# v1.4.0 更新日志 - 副标题解析

**发布日期：** 2024-10-21  
**类型：** 新功能  

## 🎯 核心功能

### 副标题解析
支持中文副标题解析，提升中文内容识别能力。

**功能：**
- ✅ "第X季" 解析
- ✅ "全X集" 解析
- ✅ 中文数字转换
- ✅ 阿拉伯数字支持

## 📦 依赖

**新增依赖：**
```bash
pip install cn2an
```

## 📝 使用示例

### 解析示例
```python
# "第七季" → season=7
# "第3季" → season=3
# "全12集" → total_episodes=12
# "全十二集" → total_episodes=12
# "第二季 全10集" → season=2, total_episodes=10
```
