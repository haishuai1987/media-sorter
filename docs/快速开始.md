# 快速开始指南

## 🚀 5分钟快速上手 Media Renamer v2.8.0

### 一、安装部署

#### 方法1：本地安装（推荐）

```bash
# 1. 克隆仓库
git clone https://github.com/haishuai1987/media-sorter.git
cd media-sorter

# 2. 安装依赖
pip install -r requirements.txt
pip install -r requirements-v2.8.0.txt

# 3. 启动应用
python app_v2.py

# 4. 访问应用
# 浏览器打开: http://localhost:8090
```

#### 方法2：Docker 部署

```bash
# 1. 使用 Docker Compose
docker-compose up -d

# 2. 访问应用
# 浏览器打开: http://localhost:8090
```

#### 方法3：自定义端口

```bash
# 使用命令行参数
python app_v2.py --port 8091

# 或使用环境变量
PORT=8091 python app_v2.py
```

### 二、首次使用

#### 1. 打开应用

访问 http://localhost:8090，你会看到现代化的 Web 界面。

#### 2. 界面介绍

**顶部导航栏**：
- 📦 批量处理 - 批量识别和重命名文件
- 🔍 文件识别 - 单文件识别测试
- ✏️ 批量编辑 - 高级批量编辑功能
- 📜 历史记录 - 查看处理历史
- ⚙️ 配置管理 - 保存和管理配置
- 📝 模板管理 - 管理命名模板
- 📚 识别词 - 管理自定义识别词
- 📊 统计信息 - 查看系统统计

**右上角按钮**：
- 🌍 - 切换语言（中文/English）
- ⌨️ - 快捷键帮助
- 🌙/☀️ - 切换主题

---

### 三、基本操作

#### 方式1：文本输入

1. 在"文件列表"文本框中输入文件名（每行一个）
2. 选择命名模板
3. 点击"👁️ 预览"查看结果
4. 点击"🚀 开始处理"执行重命名

#### 方式2：拖拽上传

1. 将文件拖拽到拖拽区域
2. 文件名自动填充到列表
3. 选择模板并处理

#### 方式3：导入文件列表

1. 点击"📥 导入列表"
2. 选择 TXT 或 CSV 文件
3. 文件名自动导入

#### 方式4：列表视图（拖拽排序）

1. 点击"🔄 切换视图"
2. 切换到列表模式
3. 拖拽文件项调整顺序
4. 点击删除按钮移除单个文件

### 四、实时进度

处理文件时，会自动显示实时进度：

```
正在处理...                    60%
████████████░░░░░░░░░░
The.Matrix.1999.mkv         6/10
```

- 显示当前处理的文件名
- 显示百分比进度
- 显示已完成/总数
- 完成后自动隐藏

### 五、主题和语言

#### 主题切换

- **左键点击** 🌙/☀️ 按钮：手动切换主题
- **右键点击** 🌙/☀️ 按钮：恢复自动跟随系统主题
- **默认行为**：自动跟随系统主题（亮色/暗色）

#### 语言切换

- 点击 🌍 按钮切换中英文
- 自动保存语言偏好
- 界面立即更新

---

### 六、快捷键

按 `Ctrl+/` 显示快捷键帮助面板

常用快捷键：
- `Ctrl+P` - 预览结果
- `Ctrl+Enter` - 开始处理
- `Ctrl+I` - 导入列表
- `Ctrl+E` - 导出结果
- `Ctrl+K` - 清空列表
- `Ctrl+T` - 切换主题
- `Ctrl+L` - 切换语言

### 八、命名模板

#### 电影模板

- **默认**: `电影名 (年份)/电影名 (年份) [1080p-BluRay].mkv`
- **简单**: `电影名 (年份)/电影名 (年份).mkv`
- **详细**: `电影名 (年份)/电影名 (年份) [1080p H264 AAC BluRay].mkv`

#### 电视剧模板

- **默认**: `剧名/Season 01/剧名 - S01E01 [1080p-WEB-DL].mkv`
- **简单**: `剧名/S01/剧名 - S01E01.mkv`
- **详细**: `剧名/Season 01/剧名 - S01E01 [1080p H265 DTS WEB-DL].mkv`

### 十、常见问题

#### Q: 如何更改端口？
A: 使用 `--port` 参数或设置 `PORT` 环境变量

#### Q: 如何停止应用？
A: 按 `Ctrl+C` 或关闭终端

#### Q: 支持哪些文件格式？
A: 支持所有常见视频格式（mkv, mp4, avi, mov 等）

#### Q: 如何备份配置？
A: 在"配置管理"页面导出配置为 JSON 文件

#### Q: 如何查看处理历史？
A: 点击"历史记录"标签页查看所有处理记录
