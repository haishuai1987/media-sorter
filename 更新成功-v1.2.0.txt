配置标题映射（可选）
===============================================

如果想为常用剧集配置映射，提高识别准确率：

1. 编辑文件：title_mapping.json
2. 添加你的剧集：
   {
     "你的剧集英文名": {
       "tmdb_id": TMDB_ID,
       "chinese_title": "中文名",
       "type": "tv"
     }
   }
3. 保存后自动生效（无需重启）
